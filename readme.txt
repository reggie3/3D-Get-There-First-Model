3D Get There First Model

Performs stochastic runs to determine who gets to a point first.

Star field image: Nick Risinger, 2011, http://skysurvey.org/
Planet and Sun textures: JHT's Planetary Pixel Emporium, http://planetpixelemporium.com/earth.html

Normally Distributed Random Numbers generated from port of the code located at ftp://ftp.taygeta.com/pub/c/boxmuller.c
